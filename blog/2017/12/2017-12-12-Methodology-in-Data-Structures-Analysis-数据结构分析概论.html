Methodology in Data Structures Analysis 數據結構分析概論

發表於 2017-12-12 | 分類於 Data Structures and Algorithms 數據結構與算法

筆者寫就本文時，數據結構已正式結課。

實現

常用的數據結構具體實現是由數組(Array)、鏈表(Linked List)及其他鏈式結構(Linked Structures)、及二者的混合體組成的。

數組的特點是從空間上說，佔用較少但固定的空間；從時間上說，按索引訪問修改元素、查找有序元素有優勢，刪除插入元素等改變數組長度的操作有劣勢，且面臨數組內存不足擴充數組的風險。當元素數量相對確定，訪問修改查找比刪除插入更頻繁時有效率。

鏈表的特點是從空間上說，佔用較多(指針域)但按需的空間；從時間上說，沒有擴充結構的風險。當元素數量極不確定時有效率。

因此，很多複雜數據結構採用二者的結合，各取所長。
